Simulador de Loteria

Sobre o Projeto

O Simulador de Loteria é um website desenvolvido para demonstrar a aplicação da Análise Combinatória e da Probabilidade em situações reais.

O usuário informa a quantidade total de números disponíveis e a quantidade de números apostados. O sistema calcula automaticamente o número de combinações possíveis e a probabilidade de ganhar.

---

Objetivos

* Aplicar conceitos de Análise Combinatória.
* Utilizar a fórmula da combinação simples.
* Demonstrar o uso de fatoriais em cálculos matemáticos.
* Desenvolver uma interface interativa utilizando HTML, CSS e JavaScript.

---

Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

Conceitos Matemáticos

O projeto utiliza a fórmula da combinação:

C(n,p) = n! / [p!(n-p)!]

Onde:

* n = quantidade total de números disponíveis
* p = quantidade de números escolhidos

Exemplo:

C(60,6) = 50.063.860

Isso significa que existem 50.063.860 apostas possíveis diferentes em uma loteria com 60 números, escolhendo 6.

---

Funcionalidades

* Inserção do total de números disponíveis.
* Inserção da quantidade de números apostados.
* Cálculo automático das combinações possíveis.
* Exibição da chance de ganhar.
* Exibição da probabilidade em porcentagem.
* Interface simples e intuitiva.
